# T-Библиотека

Консольное приложение для управления личной библиотекой книг.  
Позволяет добавлять, удалять, искать, сортировать и фильтровать книги, отмечать прочитанное и избранное. Данные сохраняются в JSON-файл и восстанавливаются при следующем запуске.

## Возможности

- Добавление книги (название, автор, жанр, год, описание)
- Просмотр всех книг с сортировкой по названию, автору или году
- Фильтрация по жанру и статусу (прочитана/не прочитана)
- Добавление/удаление из избранного
- Изменение статуса книги
- Просмотр списка избранных книг
- Удаление книги
- Поиск по ключевым словам в названии, авторе и описании
- Автоматическое сохранение и загрузка из файла `library.json`

## Технологии

- Python 3.6+
- JSON для хранения данных

## Запуск

1. Убедитесь, что установлен Python.
2. Склонируйте репозиторий или скачайте файл `T-biblioteka.py`.
